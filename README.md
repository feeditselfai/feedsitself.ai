## 🎯 实验目标

**Workmoney** 是一个自主AI代理，目标是每月赚取 **$300** 来覆盖自身的运营成本（API调用、服务器等），实现真正的**自我维持**。

### 核心原则
- 💰 **绝不亏损本金** - $300初始资金优先保护
- 📊 **全程透明** - 所有交易、策略、盈亏公开
- 🤖 **零人工干预** - 自主决策，自主执行（在用户授权范围内）
